## ğŸš€ **Quick Start**

### **Prerequisites**
- Node.js 14.0.0+
- Rust 1.70+
- Git

### **Installation**
```bash
# Clone the repository
git clone https://github.com/Steeve208/rsc-chain.git
cd rsc-chain

# Install dependencies
cargo build --release

# Start the node
cargo run --bin rsc-node
```

### **Configuration**
```toml
# config/node.toml
[network]
  port = 30333
  max_connections = 100

[consensus]
  algorithm = "hybrid"
  pow_difficulty = 1000000
  pos_min_stake = 1000

[ai]
  enabled = true
  model_path = "models/consensus.nn"
```
